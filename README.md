## About
Yet another [Vim](http://www.vim.org) config for Vim 7.4.

## Installation

### Clone
```sh
~ $ git clone https://github.com/johnadamson/vim_config.git
```

### Initialize
```sh
~/vim_config $ . ./init.sh
```

### Symlink
```sh
~ $ ln -s ./vim_config/.vim
~ $ ln -s ./vim_config/.vimrc
```

### Vundle
```
:BundleInstall
```
